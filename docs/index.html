前言

Overleaf是什么

https://www.overleaf.com/

简单讲,Overleaf是一个在线的LaTeX环境. 不需要在自己电脑上安装,通过网页访问即可编写LaTeX.

如果还不了解LaTeX,可以先阅读下面的链接:

LaTeX的介绍: https://liam.page/2014/09/08/latex-introduction/

当然,Overleaf提供的服务远不止此.

借助Overleaf,可以实现多人合作编辑,无缝同步进度,追踪文件修改历史.
